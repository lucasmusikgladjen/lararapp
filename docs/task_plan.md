# Task Plan: Pixel Perfect Dashboard Styling (Manual)

## Mål
Styla Dashboarden exakt efter Figma-designen genom manuell analys av egenskaper och översättning till NativeWind.

## Definition of Done (DoD)
- [ ] Dashboard-bakgrund och kort har korrekta Hex-färger från Figma.
- [ ] Border-radius och skuggor matchar designen visuellt.
- [ ] Spacing (padding/gap) följer Figmas 8px-grid (eller motsvarande).
- [ ] Typografi (vikt/storlek) är konsekvent med NativeWind-klasser.
